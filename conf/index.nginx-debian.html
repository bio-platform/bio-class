Switch Rstudio from HTTP to HTTPS

To switch Rstudio from HTTP to HTTPS, run one of the following commands of your choice:

  • "startHTTPS" - get a certificate from Let's Encrypt
  • "startHTTPSlocalCrt" - get self-signed certificate with OpenSSL (For Experienced Users Only)
    • (In Browser Allow Self Signed Certificate: button Advanced -> Add Exception / Accept the Risk and Continue)
  • To switch back to unsecured HTTP execute command "stopHTTPS"
  • Find out the current Rstudio URL using command "statusHTTPS"
